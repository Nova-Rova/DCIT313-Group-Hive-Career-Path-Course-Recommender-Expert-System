## Project summary 

an expert system that guides students in choosing career paths and related courses based on their interests, strengths, skills, and academic performance. The system uses rule-based decision-making (IF-THEN rules) to provide personalized recommendations on career paths, university courses, and certifications.
The system will ask users questions and apply predefined rules to suggest suitable career options and academic paths.
